# [pico-ssd1306-mono-graphics-lib](https://github.com/rppicomidi/pico-ssd1306-mono-graphics-lib)
A raspberry Pi Pico SSD1306 graphics library that can manage multiple displays at the same time.
This feature is important for projects that use one display per channel like [pico-mc-display-bridge](https://github.com/rppicomidi/pico-mc-display-bridge).
It integrates well with [pico-mono-ui-lib](https://github.com/rppicomidi/pico-mono-ui-lib).
