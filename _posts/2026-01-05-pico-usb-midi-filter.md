# [pico-usb-midi-filter](https://github.com/rppicomidi/pico-usb-midi-filter)
This project allows you to modify the MIDI data stream between a USB MIDI device and a USB Host.
It is a fixed function version of [pico-usb-midi-processor](https://github.com/rppicomidi/pico-usb-midi-processor) with no UI.
