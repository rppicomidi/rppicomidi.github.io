# [pico-w-ble-midi-server-demo](https://github.com/rppicomidi/pico-w-ble-midi-server-demo)
# A Bluetooth LE MIDI 1.0 server peripheral demo for the Raspberry Pi Pico W

This project is now archived. The project was created as a proof of concept
for Bluetooth-LE MIDI on a Raspberry Pi Pico W. For better examples,
please see [ble-midi2usbhost](https://github.com/rppicomidi/ble-midi2usbhost)
and [midi2piousbhub](https://github.com/rppicomidi/midi2piousbhub).
