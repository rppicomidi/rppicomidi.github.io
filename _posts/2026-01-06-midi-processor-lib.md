# [midi-processor-lib](https://github.com/rppicomidi/midi-processor-lib)
C++ classes for processing MIDI data packets (e.g. filtering, bi-directional processing, ...).

This is [pico-usb-midi-processor](https://github.com/rppicomidi/pico-usb-midi-processor) extracted to library form for inclusion in other projects.
It is designed to exist in the same mono graphics UI and CLI environment.
