# [TinyUSB_MIDI_Device_Arduino](https://github.com/rppicomidi/TinyUSB_MIDI_Device_Arduino)
The TinyUSB_MIDI_Device_Arduino library is now available for download in the Arduino IDE via Arduino Library Manager
as version 0.4.0. Sorry the version number is so high. I repeatedly updated tags and the `library.properties` file changes
out of sync. The underlying code that actually runs is unchanged from the initial version 0.1.0.
