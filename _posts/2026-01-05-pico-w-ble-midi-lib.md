# [pico-w-ble-midi-lib](https://github.com/rppicomidi/pico-w-ble-midi-lib)
This library contains a GATT server and a GATT client for the BLE-MIDI service, MIDI 1.0 packet encoder and decoder.

Projects that use this library must also link [ring_buffer_lib]({% link _posts/2026-01-03-ring_buffer_lib.md %})
