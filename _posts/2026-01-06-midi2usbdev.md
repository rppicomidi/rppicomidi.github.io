# [midi2usbdev](https://github.com/rppicomidi/midi2usbdev)
Make a computer USB to old school 5-pin DIN MIDI interface from a Raspberry Pi Pico.

There are a lot of microcontroller boards that can do this, but this is an easy test
of the TinyUSB USB MIDI Device class implementation.
