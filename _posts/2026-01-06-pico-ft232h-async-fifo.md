# [pico-ft232h-async-fifo](https://github.com/rppicomidi/pico-ft232h-async-fifo)
Library that interfaces the FTDI FT232h chip in Async FIFO 245 mode with a RP2040
chip's PIO to stream USB 2.0 high speed data at up to 9.9MB/sec.

This library includes a trivial test example to see just have fast data can stream.
