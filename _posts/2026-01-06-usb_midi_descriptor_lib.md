# [usb_midi_descriptor_lib](https://github.com/rppicomidi/usb_midi_descriptor_lib)
TinyUSB extension to get the strings from a USB MIDI Device's interface descriptors.

The strings descriptors associated with the Input Jacks and Output Jacks can provide useful labels on a user interface.
